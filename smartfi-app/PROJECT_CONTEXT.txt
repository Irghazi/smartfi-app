## 7. FITUR UNGGULAN (Key Features) — Sudah Termasuk Fitur Baru

### A. Smart Input (AI-Powered) — Fitur Andalan!
Pengguna cukup ketik kalimat alami di satu kotak teks, lalu AI (DeepSeek) mem-parsing menjadi data transaksi.
**Contoh:** *"Makan siang warteg 25 ribu"* → `{ type: 'expense', category: 'Makanan', nominal: 25000, description: 'Makan siang warteg' }`

### B. Widget Lingkaran 2x2 (Mobile Dashboard) — FITUR BARU!
Khusus untuk tampilan HP, di bagian atas dashboard tampilkan 4 lingkaran (Doughnut Chart) dalam grid 2x2:
1. **Saldo** (Biru `#60A5FA`) — progress terhadap target tabungan.
2. **Pemasukan** (Hijau `#10B981`) — total pemasukan bulan ini.
3. **Pengeluaran** (Merah `#EF4444`) — total pengeluaran + sisa budget.
4. **Kategori Terbesar** (Ungu `#8B5CF6`) — nama kategori + persentase.

### C. Mentor AI Pribadi
AI yang memiliki konteks penuh atas seluruh data transaksi user. Fungsi: Deteksi anomali, identifikasi kategori terbesar, prediksi saldo, dan 3 saran hemat.

### D. Pengingat Harian (Reminder) — FITUR BARU!
User bisa mengatur jam pengingat (1x sehari) di halaman Pengaturan.
- **Notifikasi Browser (Web Push):** Muncul di HP di jam yang ditentukan.
- **In-App Banner:** Muncul saat user buka aplikasi jika hari ini belum ada transaksi.
- **Alasan 1x sehari:** Membangun kebiasaan, menghormati waktu user, menghindari spam.

### E. PWA (Progressive Web App) — Installable — FITUR BARU!
SmartFi bisa diinstal ke layar utama HP seperti aplikasi native.
- **Android (Chrome):** Klik "Pasang Aplikasi" di address bar.
- **iOS (Safari):** Klik "Share" → "Add to Home Screen".
- **Keuntungan:** Fullscreen, icon di layar utama, update tanpa Play Store.

### F. Manajemen Transaksi & Kategori
CRUD lengkap + filter + search.

### G. Laporan & Ekspor (PDF/Excel)
Filter tanggal + jenis laporan, ekspor ke PDF atau Excel.
